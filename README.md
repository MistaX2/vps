## 📌 About

This project provides an automated installation script to quickly set up a VPS environment with required packages, configurations, and optimizations.

It simplifies server deployment with a single command.

---

## ⚡ Features

* 🔧 Automatic package installation
* 🔐 Basic security configuration
* 🌐 Network optimization
* 🛠 System update & cleanup
* 📦 Service auto configuration

---

## 🖥 Supported Systems

* Ubuntu 20.04+



---

## 📥 Installation

Run the following command on your VPS:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/MistaX2/vps/refs/heads/main/install) 
```

⚠️ **Important:**
Always review the script before running it on production systems.

---

## 🔍 Manual Installation (Safer Method)

```bash
bash <(curl -Ls https://raw.githubusercontent.com/MistaX2/vps/refs/heads/main/install) 
nano install
bash install
```
